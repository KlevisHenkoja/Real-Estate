MERN technology was used in this project.
after downloading this project, you must unzip it.
Open the project in a code editor:
1. Open the terminal
2. Go to the "client" folder by typing the command "cd client" and install the React package
3. Then go back by typing the command "cd .."
4. install the backend package by typing "git init -y"
5. Go to the api folder "cd api"
6. we start installing all the packages we need in the backend
   a. Install the express package with the command "npm install express"
   b. Install the MongoDB package "npm install mongoose"
   c. Install cookie parser "npm install cookie-parser"
   d. Install bcryptjs to encrypt the password "npm install bcryptjs"
   e. Install json web token "npm install jsonwebtoken"
7. Go to the frontend folder "cd .." and then "cd client"
8. Install the packages we need in the frontend:
  a. Install react router dom "npm i react-router-dom"
  b. Install TailwindCss "npm install -D tailwindcss"
  c. Install reduxjs "npm install @reduxjs/toolkit"
  d. Install react redux "npm install react-redux"
  e. Install Firebase "npm install firebase"
  p. Install Swiper "npm i swiper"
  g. Install react icons "npm install react-icon"
